#  kRadial

isotropic wavenumber dimension


---

## Discussion

Creates a radial wavenumber axis,

$$
k_r \equiv \sqrt( k^2 + l^2)
$$

with spacing set by the largest gap in the gridded radial wavenumber. See `radialWavenumberAxis`.
